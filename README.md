# VLSI Task 3 - Verilog HDL
This repository contains the Verilog HDL implementation and simulation of fundamental sequential digital circuits.

## Implemented Circuits
- D Flip-Flop
- JK Flip-Flop
- 4-bit Register
- 4-bit Counter

## Tools Used
- Verilog HDL
- ModelSim Intel FPGA Starter Edition 2020.1

## Repository Contents
Each circuit includes:
- Verilog design (.v)
- Testbench (.v)
- Simulation waveform

The circuits were simulated and verified using ModelSim to ensure correct sequential operation and waveform analysis.

## Learning Outcomes
- Sequential circuit design using Verilog HDL
- Flip-Flop implementation
- Register design
- Binary Counter design
- Testbench development
- Functional simulation and waveform verification
